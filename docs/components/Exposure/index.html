使用​

基本使用​

Props​

参数类型默认值
idstring-

必传，设置曝光监听的id属性

children?JSX.Element | JSX.Element[]无

非必传，自定义子元素的dom

delay?number500

非必传，回调的延迟时间，默认500毫秒。

onShow?(id?: string) => void | (() => Promise<void>)无

非必传，曝光时的回调函数，id为组件传入的id。

onHide?(id?: string) => void | (() => Promise<void>)无

非必传，隐藏（不再曝光）时的回调函数，id为组件传入的id。

area?'part' | 'whole''whole'

非必传，是否是整个区域曝光，’part’表示部分区域曝光就可触发回调函数，’whole’表示整个区域曝光就可触发回调函数，默认是 ‘whole’。

debug?boolean无

非必传，是否进入debug模式，默认是false

type?'vertical' | 'horizontal''vertical'

非必传，debug模式会展示给需要曝光的元素增加一条红线，方便调试 ‘vertical’表示水平红线，’horizontal’表示垂直红线，默认是’vertical’
