yuan2fen

金额元转分，添加separator=true时，将格式化金额

使用例子​

import { yuan2fen } from '@credit-biz/utils'

console.log(yuan2fen(100, true));  //'100.00'
console.log(yuan2fen(100));  //100

参数​

参数说明默认值
amt金额元-
separator是否格式化添加逗号-

返回值​

string | umber
